To access the UI:

    1. kubectl port-forward svc/paxful-email-service 8025:8025
    2. Then access in browser: http://localhost:8025

To access the WebApp if you are using ClusterIP:

    1. kubectl port-forward svc/paxful-test-solution-service 8080:80
    2. Then access in browser: http://localhost:8080

To access Prometheus from outside the cluster execute the following commands:

    1. kubectl port-forward --namespace default svc/prometheus-kube-prometheus-prometheus 9090:9090
    2. Then access in browser: http://127.0.0.1:9090


To access grafana:

    1. kubectl port-forward svc/grafana 3000:3000
    2. Then access in browser: http://localhost:3000
    3. If it is the first time you are running grafana:
        1. Add Prometheus datasource
        2. Import PostgreSQL Dashboard with ID: 9628
